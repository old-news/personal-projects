Like a RNN, but some connections are set to 0. The idea is that it accurately simulates a brain.
This was inspired by this YouTube video by davidrandmiller: https://www.youtube.com/watch?v=N3tRFayqVtk&t=1751s

The creatures in my program are trying to get to the left side of the screen. We simulate evolution by selecting those on the left side and then creating the next generation based on those.
This is my favorite project so far. It took about 13 hours to code up.
